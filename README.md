# Ghaith – Smart Humanitarian Relief Tent

This project was developed as part of the Embedded Systems course.

## Description
A smart humanitarian tent designed to assist displaced families by monitoring temperature and environmental conditions and controlling a heating system automatically.

## Technologies
- Arduino Uno
- DS18B20 Temperature Sensor
- Relay Module
- Servo Motor
- Rain Sensor

## Features
- Automatic heater control
- Temperature-based hysteresis control
- Safety isolation using relay
